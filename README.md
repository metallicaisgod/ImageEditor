# ImageEditor
Программа редактирования изображений:
 -обрезка, поворот;
 -добалвение текста, стикеров, смайликов;
 -рисование поверх картинки (фигуры, стреклки, кисть);
 -наложение фильтров.
Изображение можно взять как с камеры, так и из галереи.
